Der Lichtschalter...

-Szenario-

Der Bewohner einer einsamen Behausung schaltet normalerweise auf dem Weg ins Bett jeden Abend das Licht an. Als er eines Morgens Radio hörte, sürzte er sich ins Meer.

Aufgabenstellung

  1. Kontaktieren Sie den Chatbot-Support von Handytarife2000.
  2. Beschreiben Sie Ihr Anliegen
    (z.B. "Meine Handyrechnung war diesen Monat sehr hoch!!")
  3. Geben Sie Ihre Kundennummer, sowie die Rechnungsnummer an.

    4. Kunden-Nr.:         Kunde123
    Rechnungs-Nr.:   Rechnung123

  4. Finden Sie heraus, warum die Rechnung diesen Monat höher ist als sonst.
  5. Akzeptieren Sie die hohen Kosten in diesem Monat nicht.
  6. Füllen Sie im Anschluss an das Gespräch den Fragebogen aus.
    Den Link dazu erhalten Sie vom Chatbot am Ende der Unterhaltung.
